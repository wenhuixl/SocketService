unicode引用

unicode是字体在网页端最原始的应用方式，特点是：

  • 兼容性最好，支持ie6+，及所有现代浏览器。
  • 支持按字体的方式去动态调整图标大小，颜色等等。
  • 但是因为是字体，所以不支持多色。只能使用平台里单色的图标，就算项目里有多色图标也会自动去色。

注意：新版iconfont支持多色图标，这些多色图标在unicode模式下将不能使用，如果有需求建议使用symbol的引用方式

unicode使用步骤如下：

第一步：拷贝项目下面生成的font-face

@font-face {
  font-family: 'KLP-iconfont';
  src: url('KLP-iconfont.eot');
  src: url('KLP-iconfont.eot?#iefix') format('embedded-opentype'),
  url('KLP-iconfont.woff') format('woff'),
  url('KLP-iconfont.ttf') format('truetype'),
  url('KLP-iconfont.svg#iconfont') format('svg');
}

第二步：定义使用iconfont的样式

.KLP-iconfont{
  font-family:"KLP-iconfont" !important;
  font-size:16px;font-style:normal;
  -webkit-font-smoothing: antialiased;
  -webkit-text-stroke-width: 0.2px;
  -moz-osx-font-smoothing: grayscale;
}

第三步：挑选相应图标并获取字体编码，应用于页面

<i class="KLP-iconfont">&#x33;</i>

实际情况中"KLP-iconfont"（font-family）需要修改为你项目下的font-family。可以通过编辑项目查看，默认是"KLP-iconfont"。
